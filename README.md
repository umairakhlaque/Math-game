# 🦸 Math Superhero Academy

An educational maths game for **Class 1–3 students** (UK National Curriculum KS1/KS2). Defeat funny villains using your maths powers!

## 🎮 Features

- **15 missions** across 3 worlds (Class 1 → Class 2 → Class 3)
- **Funny villains** with comic-book speech bubbles
- **10 question types**: counting, addition, shapes, odd/even, money, times tables, fractions, measurement, time, perimeter, word problems
- **Infinite unique questions** — procedurally generated every game
- **Gamification**: stars, streaks, HP bars, badges, confetti
- **11 badges** to unlock
- **Web Audio sound engine** — no external audio dependencies
- **Professor Hoot hint system** 🦉
- Dark comic-book aesthetic with smooth animations

## 🌍 Worlds

| World | Level | Missions |
|-------|-------|----------|
| 🌟 Recruit Hero | Class 1 | Count Castle, Addition Alley, Shape Village, Odd Even Kingdom, Money Mountain |
| 🚀 Rising Hero | Class 2 | Hundred Hills, Times Table Tower, Fraction Forest, Measurement Maze, Time Temple |
| ⚡ Math Superman | Class 3 | Times Table Titan, Division Dungeon, Fraction Fortress, Perimeter Palace, Word Problem Volcano |

## 🚀 Getting Started

```bash
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173)

## 🏗️ Build

```bash
npm run build
```

## 🛠️ Tech Stack

- **React 18** with hooks
- **Vite** for fast dev/build
- **Web Audio API** for sound effects
- **CSS animations** (no external animation libraries)
- Pure inline styles — no CSS framework needed

## 📚 Curriculum Coverage

- **Class 1**: Counting to 100, addition within 20, 2D shapes, odd/even, coins
- **Class 2**: Addition to 100, ×2/5/10 tables, halves & quarters, measurement, analogue time
- **Class 3**: All times tables (2–12), division, fraction comparison, perimeter, multi-step word problems
